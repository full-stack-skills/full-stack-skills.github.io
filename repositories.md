## 状态含义

- **公开**：任何人都可以读取仓库；不自动代表已发布稳定制品。
- **Private 预览**：项目真实存在，但只对有权限的成员开放，外部访问链接可能返回 404。
- **已归档**：保留历史，不应作为新项目默认依赖。

## 使用前检查

1. 阅读目标仓库 README 与 release 页面。
2. 确认许可证、运行时基线和稳定性声明。
3. 在自己的环境执行该仓库门禁，不用本目录代替验证。
